## [0.8.5] — 2026-07-14

### Sprint 8.5 — Developer infrastructure

### Added

- Engineering documentation for roadmap, architecture, data, APIs, design,
  content, deployment, testing, and change history
- Canonical domain types and constants
- Shared service boundaries for market, authentication, CMS, payments,
  notifications, analytics, validation, utilities, and configuration

- Consolidated duplicated email normalization and validation
- Consolidated display-label and date formatting
- Centralized navigation, routes, markets, membership benefits, and color tokens
- Added CMS, auth, and payment service facades
- Preserved the root layout and current pages as Server Components
